# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-08

### Added

- **Company** resource: Create, Get, Get Many, Update, Delete
- **Person** resource: Create, Get, Get Many, Update, Delete
- **Opportunity** resource: Create, Get, Get Many, Update, Delete
- **Note** resource: Create, Get, Get Many, Update, Delete
- **Task** resource: Create, Get, Get Many, Update, Delete
- Full cursor-based pagination with "Return All" support
- Advanced filtering UI (name search, date ranges, stage, company links)
- Ordering support (created at, name, updated at)
- AI Agent compatibility (`usableAsTool: true`)
- Comprehensive error handling with actionable messages
- 17 unit tests + 7 integration tests
- GitHub Actions publish workflow with npm provenance
- Zero runtime dependencies
